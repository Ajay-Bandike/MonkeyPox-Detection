# Monkeypox Detection Project

This project predicts Monkeypox disease using Machine Learning and Django.

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- Machine Learning

## Features
- User Login and Registration
- Disease Prediction
- Dataset Analysis
- Frontend and Backend Integration

## How to Run

1. Clone repository
2. Install requirements
3. Run server

pip install -r req.txt
